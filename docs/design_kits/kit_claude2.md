# Kit: Claude2 Editorial Grid

## Summary
Editorial, high-contrast, paper-and-ink look. Heavy rules, hard borders, tight mono labels.

## Fonts
- Heading: Bebas Neue
- Body: IBM Plex Sans
- Label: IBM Plex Mono

## Palette
- Background: #f2ede4 (paper)
- Background alt: #e5dfd3 (paper dark)
- Ink: #0d0d0d
- Accent 1: #d4380d (red)
- Accent 2: #2563a8 (blue)
- Muted text: #6e6760
- Rules: rgba(13,13,13,0.12)

## Layout
- Use grid with visible dividers and border lines
- Section separators: 2px solid ink
- Cards are square-ish, sharp corners, no soft shadows

## Navigation
- Sticky bar with 2px ink borders
- Logo left with mono superscript accent
- Links uppercase mono, tight tracking
- CTA as full-height block with solid accent background

## Hero
- Split grid with strong typographic headline
- Labels in mono with line rules
- Use accent color to highlight 1-2 words

## Services
- Use row-based list with strong dividers
- Hover inverts to ink background with paper text

## Plan Search Input
- White/paper background, 2px ink border
- Large padding, mono label above
- Search icon inside input, text in ink

## Plan Dropdown Results
- Full-width cards
- 2px ink border, square corners
- Uniform width, no auto-shrink
- Left aligned text, arrow on right

## Strategy Carousel
- Horizontal, one card per view
- Card: paper background, ink border, accent icon circle
- Use mono label + large heading + bullet list

## Contact Form
- Two-column layout with strong grid
- Inputs with 2px ink border, paper background
- Labels uppercase mono with tracking
- Submit button solid accent with ink text
