# Kit: Sample Corporate Blue

## Summary
Clean corporate blue. Soft shadows, rounded cards, hero image with gradient overlay.

## Fonts
- Heading: Playfair Display or similar serif
- Body: Inter or similar sans
- Label: Inter or mono for small uppercase labels

## Palette
- Primary: #0f172a (slate)
- Secondary: #2563eb (blue)
- Background: #ffffff
- Muted text: #6b7280
- Borders: #e5e7eb

## Layout
- Generous vertical spacing, centered section headers
- Cards with rounded corners and soft hover
- Thin borders, subtle shadows

## Navigation
- Fixed, white with blur and thin border
- Serif logo with colored accent word
- CTA as filled button

## Hero
- Full-bleed image with left-to-right gradient overlay
- Eyebrow pill with light border
- CTA large with arrow icon

## Services
- 3-column cards, hover background tint
- Icon inside rounded square

## Plan Search Input
- Rounded, 2px border, white background
- Icon inside, large padding
- Focus blue ring or border

## Plan Dropdown Results
- Full width card list
- Rounded, light border, subtle shadow
- Consistent width, left aligned text, arrow right

## Strategy Carousel
- Horizontal scroll with snap
- Card fill in soft tints (green, red, blue)
- Rounded corners, light border and shadow

## Contact Form
- Split card: dark left panel, light form on right
- Inputs with rounded corners and soft border
- Primary CTA full width
