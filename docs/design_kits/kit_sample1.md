# Kit: Sample1 Premium Gradient

## Summary
Premium navy and gold with glowing gradients. Glassy cards and rich contrast.

## Fonts
- Heading: Playfair Display
- Body: Inter
- Label: Inter uppercase or mono

## Palette
- Navy: #0b1c3a
- Gold: #f3b34c
- Blue accent: #2563eb
- Background: #f8fafc
- Muted text: #64748b

## Layout
- Centered headers, large type
- Soft gradient section backgrounds
- Rounded 2xl cards with deep shadow

## Navigation
- Transparent or light bar over hero
- Logo serif with gold accent
- CTA button filled navy or gold

## Hero
- Large headline, airy spacing
- Optional background image or soft gradient
- Pill eyebrow in gold

## Plan Search Input
- Large rounded input with inner shadow
- Search button inside on right
- Focus ring in blue

## Plan Dropdown Results
- Rounded 2xl card list inside a shadowed panel
- Subtle hover tint
- Uniform width with right chevron

## Strategy Carousel
- One-card viewport, swipe/scroll
- Cards with colored icon badges
- Tabs style can be represented as card headers or label bands

## Contact Form
- Dark gradient section with blurred glow
- Form inside glass card with white border
- Inputs light-on-dark with clear label contrast
