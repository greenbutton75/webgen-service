  /* Raw channels for hsla() construction */
  --primary-h: HUE; --primary-s: 55%; --primary-l: 22%;
  --gold-h: GOLD_HUE; --gold-s: 62%; --gold-l: 52%;
}

For gold accent: if primary hue < 60 or > 300 → use 42 as GOLD_HUE.
Otherwise: GOLD_HUE = (HUE + 40) % 360.

⚠️ CRITICAL CSS RULE — NEVER DO THIS:
  rgba(var(--primary), 0.5)    ← BROKEN, CSS vars with hsl() cannot go inside rgba()
  rgba(var(--gold), 0.2)       ← BROKEN

✅ CORRECT — use hsla() with channel vars OR hardcoded values:
  hsla(var(--primary-h), var(--primary-s), var(--primary-l), 0.95)   ← correct
  hsla(42, 62%, 52%, 0.2)                                              ← correct
  color-mix(in srgb, var(--gold) 20%, transparent)                    ← correct (modern)
